# 清华大学实践支队赴四川省平昌县第二中学支教/社会学项目

版本：v0.1  
日期：2026-06-18  
项目定位：一周支教 + 认知启发 + 县域高中学生成长困境调研 + 可交接公共议题方案

## 0. 总控启动

### 项目触发信息

支队将前往四川省平昌县第二中学开展为期一周支教。用户希望项目不止停留在“讲几节课”，而是能够对学生认知产生真实影响，并在接触学生现实困境后，提炼公共问题，设计调研，横向询问其他支队是否存在类似现象，最终形成有故事、有证据、有解决机制的社会实践成果。

### 初始限制

1. 本阶段只做启动前治理、准备期设计和执行前闸门，不把任何未调研结论写成事实。
2. 平昌二中学生真实困境尚未进入田野验证，所有问题均为“待验证问题假设”。
3. 涉及未成年人访谈、夜谈、住宿观察、影像记录时，必须遵守学校许可、监护人授权、学生知情同意、隐私保护和非封闭一对一原则。
4. 一周支教时间很短，项目必须优先验证最关键的不确定性，而不是追求“大而全”。

### 阶段顺序

按照《复杂项目启动前置治理协议 v3 核心版》连续推进阶段 0-9：材料 intake、情景识别、发散、外部方法搜索、主链收束、Goal、Plan、评分、最小闭环、执行前闸门。

### current_basis

本项目当前依据为：用户目标、前置治理协议、公开可核验的清华支教/实践案例、平昌二中公开学校信息、未成年人研究伦理与参与式行动研究方法。学生真实需求、学校内部情况、家长态度、班级差异、住宿生活和其他支队比较数据尚未进入 current_basis。

### 情景识别

主类型：教育公益型社会实践项目。  
副类型：县域高中学生成长困境的探索性社会学调研、短周期教育干预、公共议题提炼与解决方案设计。

### 当前最大不确定性

我们不知道平昌二中学生真正卡住的是什么。可能是学习方法，也可能是升学信息、家庭经济预期、县域资源分层、自我效能、心理压力、艺体路径、手机与睡眠、寄宿生活、教师期待、同伴文化，或这些因素的组合。

### 现在最不该急着做什么

不该急着定一个“宏大主题”，也不该把清华同学的经验宣讲当作核心成果。短期支教最容易令人感动，但最容易留下的也只是一次性兴奋。准备阶段要先设计一套能把“感动”转成“可验证问题”的机制。

### 必须澄清的问题

1. 支队人数、成员专业、可授课科目、性别比例、是否有心理/教育/社科背景。
2. 学校愿意开放的对象：高一、高二、高三、初中、艺体生、住宿生、网络班、普通班。
3. 学校允许的形式：班会、正课、晚自习、宿舍走访、教师座谈、家长电话访谈、问卷。
4. 是否可与其他清华支队或支教组织建立横向比较问卷。
5. 是否有后续半年陪伴资源，例如线上答疑、书信、校友分享、导师制。

## 2. 高质量发散：候选公共议题

以下均为待验证方向，不是结论。

### 方向 A：县域高中生的“未来可达性”困境

核心问题：学生是否不是没有理想，而是缺少“从今天到未来”的可操作路径图。  
价值：能把清华支教的优势从“讲清华多好”转为“帮助学生建立可达路径”。  
所需资源：学生问卷、路径地图工作坊、升学/专业/职业资料包、校友或大学生访谈。  
最大风险：容易变成励志宣讲，忽略现实约束。  
可验证方式：前后测“我知道下一步怎么做”的变化；访谈学生是否能说出具体行动。  
依据：inference + external_method。

### 方向 B：升学信息差与专业想象贫困

核心问题：学生是否缺少关于专业、城市、大学生活、招生路径、奖助学金、艺体/强基/综评等信息的可信解释。  
价值：清华学生能提供稀缺的一手经验，但要避免“名校滤镜”。  
所需资源：专业地图、志愿填报基础资料、奖助学金说明、不同层次高校路径案例。  
最大风险：信息过载，或只服务成绩较好的学生。  
可验证方式：学生能否完成一张“我感兴趣的 3 个方向 + 要补的 3 个信息”。  
依据：inference + material_fact（平昌二中有艺体、网络班、导师制等公开信息）+ external_method。

### 方向 C：压力、睡眠、手机与寄宿生活的日常治理

核心问题：学生的学习行动是否被睡眠、手机、晚自习节奏、宿舍关系、情绪压力共同影响。  
价值：能把“学习方法”落到一天的生活结构中。  
所需资源：匿名时间日记、宿舍/夜谈圆桌、教师访谈、非侵入式观察。  
最大风险：触及隐私和学校管理敏感点。  
可验证方式：匿名时间使用统计 + 学生共同提出“可改变的一天”。  
依据：assumption + external_method。

### 方向 D：县域高中里的“被看见”与自我效能

核心问题：学生是否感觉自己被老师、家庭、同伴或外部世界看见；这种被看见感是否影响学习信念。  
价值：对认知影响更深，不只给方法，还给身份确认。  
所需资源：故事访谈、心愿瓶、成长信件、同伴互访、积极反馈活动。  
最大风险：短期温情难以持续。  
可验证方式：学生能否写出“我有一个值得被认真对待的问题/计划”。  
依据：external_method + inference。

### 方向 E：艺体特色与多元升学路径

核心问题：艺体学生、普通文化课学生、网络班学生的路径资源是否不同，是否存在被忽略的群体。  
价值：平昌二中公开资料显示艺体和网络班是学校特色，适合做差异化调研。  
所需资源：分组访谈、艺体生专场、教师访谈、毕业生路径收集。  
最大风险：支队对艺体升学政策不熟。  
可验证方式：识别 2-3 类学生路径画像和资源缺口。  
依据：material_fact + inference。

### 方向 F：短期支教如何从一次活动变成可持续资源

核心问题：一周后，学生是否还能持续获得可信反馈、资料和陪伴。  
价值：回应优秀支教案例中的长期化、资源化经验。  
所需资源：线上答疑、书信、资源包、下一届支队交接、校内社团/SAEPA 对接。  
最大风险：队员返校后维护不足。  
可验证方式：建立 3 个月最小陪伴机制，并记录使用率。  
依据：external_method。

### 推荐主候选

以 A+B+D 为主链，C+E 作为田野发现通道，F 作为成果转化机制。

项目故事开头可以这样设置：

> 我们原以为，县域高中生最需要的是几节来自清华的好课和一次被名校点燃的梦想。但在出发前，我们先怀疑这个判断：学生真正缺的也许不是梦想，而是把梦想拆成路径的能力、可信的信息来源，以及在现实约束下仍相信自己可以行动的经验。于是，我们把一周支教设计成一次共同调查：让学生告诉我们，他们的未来在哪里被卡住。

### 推荐组合

1. 用清华书脊计划/烛光计划确定“一周支教的活动骨架”：课程、辅导、沙龙、调研并行。
2. 用清华研支团/SAEPA 确定“优秀支教的长期逻辑”：前期训练、真实学情、课后陪伴、资源化交接。
3. 用 YPAR + Photovoice 设计“学生参与式发现问题”：让学生共同命名困境。
4. 用未成年人伦理资料约束“夜谈、宿舍观察、访谈、影像记录”边界。

## 4. 主链收束与阶段路由

### 一句话主链

面向平昌二中高中生，在一周短期支教限制下，以“未来可达性”为核心矛盾，通过课程启发、参与式调研、夜谈/沙龙、路径地图和横向比较，识别县域高中生从理想到行动之间的公共障碍，并用前后测、学生作品、访谈证据和三个月后续机制验证支教是否真正改变了学生的认知与行动计划。

### 通俗版解释

我们不是去告诉学生“清华很好，你们要努力”，而是和他们一起弄清楚：他们想去哪里，为什么觉得去不了，真正缺的是信息、方法、信心、陪伴、家庭支持还是生活节奏。支队一周内要做的，是把这些卡点看见、说清、验证，并给学生留下能继续使用的路径工具。

### 专业版解释

本项目将短期教育志愿服务转化为探索性参与式行动研究。核心变量不是单节课满意度，而是学生对未来路径的可理解性、可达性感知、自我效能、信息获得能力和行动计划具体度。通过混合方法收集数据：匿名问卷、路径地图、焦点小组、教师访谈、学生夜谈、时间日记、Photovoice 轻量版和跨支队比较。干预模块包括认知启发课、大学/专业/职业地图、学习策略诊所、同伴支持和后续资源包。

### 保留范围

1. 平昌二中学生成长困境的探索性调研。
2. 一周支教课程与调研融合。
3. 面向学生的可操作资源：路径地图、信息包、行动清单。
4. 面向学校和后续支队的匿名化问题报告。
5. 与其他支队进行横向比较，判断问题是否具有公共性。

### 边界

1. 不做心理治疗，不收集创伤细节。
2. 不承诺提升成绩或改变升学结果。
3. 不做未经许可的拍摄、录音、宿舍进入或家长信息收集。
4. 不把个体故事包装成普遍结论。
5. 不以清华身份制造单向崇拜。

### backlog

1. 艺体路径专题。
2. 网络班/普通班/重点班差异专题。
3. 家庭经济与升学风险评估专题。
4. 手机、睡眠、注意力专题。
5. 乡村/县城学生空间迁移与身份认同专题。

## 5. Goal

### 系统使命

把一次清华支队的一周支教，做成一次对县域高中生“未来可达性”的共同发现、共同命名和共同修复：既让学生获得可操作的认知改变，也让支队提炼出有证据、有故事、有公共价值的教育议题。

### 第一性原理

1. 学生不是被动接受帮助的对象，而是理解自身处境的共同研究者。
2. 真正有意义的支教，不是展示外部优越资源，而是帮助学生把现实约束转化为可行动路径。
3. 公共议题必须从多个个体故事中被谨慎抽象出来，并通过横向比较检验其共性。
4. 一周项目的价值不在于解决所有问题，而在于找准一个关键问题，留下一个可延续机制。

### 核心主链

未来可达性：学生能否从“我想要一个更好的未来”走到“我知道未来有几条可能路径、每条路径需要什么、我接下来一周/一月能做什么、我可以向谁求助”。

### 不变量

1. 所有结论必须区分事实、推断、假设和待确认。
2. 所有学生材料必须匿名化。
3. 所有课程必须服务于调研主链或学生真实行动。
4. 所有故事必须尊重学生，不把困境戏剧化。
5. 所有成果必须可交接给学校、下一届支队或后续志愿者。

### 边界

项目不进入医疗、心理治疗、家庭纠纷干预、个案资助筛选、成绩承诺和未经批准的个人信息收集。

### 证据等级

| 等级 | 说明 | 本项目例子 |
| --- | --- | --- |
| E0 | synthetic 模板 | 本文件中的样例访谈提纲 |
| E1 | 公开背景材料 | 平昌二中招生简章、清华案例 |
| E2 | 现场匿名学生材料 | 问卷、路径地图、便利贴 |
| E3 | 多方交叉验证 | 学生 + 教师 + 家长/毕业生 + 其他支队 |
| E4 | 后续追踪 | 1 个月/3 个月后学生行动反馈 |

### 评价维度

1. 对学生：路径清晰度、自我效能、信息获得、行动具体度、被尊重感。
2. 对学校：是否提供有用的匿名反馈和可复用资源。
3. 对支队：是否形成公共问题、证据链和可持续机制。
4. 对社会实践评优：是否有故事线、方法论、数据、解决方案和反思边界。

### 阶段门

1. 出发前：完成学校沟通、伦理边界、工具包、课程试讲。
2. 到校第 1 天：完成基线问卷和教师访谈，确认调研样本。
3. 第 3 天：形成问题初判，决定主线是否需要回退。
4. 第 5 天：完成最小干预和学生作品。
5. 第 7 天：完成校内反馈、学生告别、后续机制启动。
6. 返校后 2 周：完成初版报告和资源包。
7. 返校后 1-3 个月：完成追踪和公共议题定稿。

### 终止条件

1. 学校不允许任何调研：项目降级为“课程 + 匿名反馈 + 资源包”。
2. 伦理条件无法满足：取消访谈、夜谈、影像等方法。
3. 现场发现学生负担过重：减少调研，保留最小问卷和开放信箱。
4. 无法建立后续承接：不承诺长期陪伴，只交付学校资源。

### 自我纠偏规则

每天晚间复盘问三件事：今天我们是在理解学生，还是在展示自己；今天的新发现是否改变了主链；明天的安排是否减少而不是增加学生负担。

## 6. Plan：准备期任务

为什么做：防止支队自嗨，确保所有调研形式合规。

风险：学校只接受授课。  
停止条件：校方拒绝调研时，降级为匿名课堂活动。

输出：角色表。

风险：样本偏差大。

人看产物：横向比较表。  
机器看产物：comparison_matrix。

横向比较 10 题：

1. 学生最常问大学生的问题是什么？
2. 学生最常表达的未来困惑是什么？
3. 学生是否知道专业与职业的关系？
4. 学生是否了解奖助学金和家庭经济支持路径？
5. 学生的压力主要来自成绩、家庭、同伴、手机、睡眠还是未来不确定？

9. 一周后支队能留下什么资源？
10. 你们认为这个学校最值得继续追问的公共问题是什么？

停止条件：任何队员无法遵守时取消相关方法。  
与 Goal 的关系：没有伦理就没有可信项目。  
人看产物：伦理清单。  
机器看产物：ethics_protocol。

机器看产物：followup_plan。

## 8. 最小闭环

### 测试对象

验证“未来可达性”是否是平昌二中学生中值得追问的关键问题，以及支队一周内是否能让学生把未来困惑转化为更具体的行动计划。

### 当前依据

用户目标、清华案例、平昌二中公开背景、参与式调研方法。

### 不作为依据

任何尚未发生的学生故事、未经允许的个体信息、队员主观印象、宣传性照片、孤立感人故事。

### 闭环类型

方案可行性闭环 + 轻量真实结果闭环。

### 验证模块表

| 模块 | 输入 | 输出 | 接口 | 验证方式 | 失败边界 | 能证明什么 | 不能证明什么 |
| --- | --- | --- | --- | --- | --- | --- | --- |

### 能证明什么

1. “未来可达性”是否是现场学生反复表达的问题。

4. 该问题是否在其他支队场域中也出现。

### 不能证明什么

1. 支教提高了成绩。
2. 支教解决了学生心理问题。
3. 平昌二中所有学生都具有同样困境。

# 准备期作战手册

## 一、项目标题候选

1. 《看见可达的未来：县域高中生未来路径与行动能力调研》
2. 《从“想去远方”到“知道下一步”：平昌二中未来可达性支教实践》
3. 《不只是一周课堂：县域高中生升学信息差、自我效能与路径支持研究》
4. 《把梦想拆成路：清华实践支队赴平昌二中支教与参与式调研》

推荐标题：**《把梦想拆成路：清华实践支队赴平昌二中“未来可达性”支教与调研项目》**

## 二、故事型公共议题设计

### 起点问题

我们出发前先提出一个谨慎怀疑：县域高中生真的最需要“被点燃梦想”吗？还是他们早已有梦想，只是不知道如何把梦想拆成路径？

### 预设误判

1. 我们可能以为学生缺的是名校视野，但他们可能缺的是本地可执行路径。
2. 我们可能以为学生缺学习方法，但他们可能被睡眠、手机、家庭期待、信息差共同困住。
3. 我们可能以为一次励志宣讲会带来改变，但学生真正需要的是后续可询问的人和资料。
4. 我们可能以为“清华经验”普适，但县域高中学生面对的是不同的资源约束。

### 田野中的故事线

第一幕：我们带着课程和热情来到平昌二中，以为要做的是“把大学带到他们面前”。  
第二幕：问卷、夜谈和路径地图告诉我们，学生的问题不一定是没有目标，而是目标到行动之间断裂。  
第三幕：我们把问题拆成信息差、行动差、自我效能差、支持网络差，并横向询问其他支队。  
第四幕：我们发现其中一部分不是平昌特例，而是县域高中学生普遍可能面对的“未来可达性”问题。  
第五幕：我们留下路径地图、资料包、后续答疑机制和学校反馈报告，把一周支教变成长期资源入口。

### 最终公共议题表达

县域高中学生的成长困境，可能不只是教育资源数量不足，而是“未来可达性”不足：他们看得到更好的未来，却缺少把未来拆解为信息、路径、行动和支持网络的机制。短期支教的公共价值，不应止于激励，而应成为帮助学生建立未来路径能力的低成本连接器。

## 三、调研问题树

### 世界问题：现实中到底发生了什么

1. 学生如何描述自己的未来？
2. 学生最想知道但无人可问的问题是什么？
3. 学生对大学、专业、职业、城市、奖助学金了解多少？
4. 学生一天的时间和压力如何分布？
5. 学生觉得自己最缺哪类支持？
6. 教师眼中的学生困境与学生自述是否一致？

### 经验规格：什么结果算好

1. 学生能说出至少 2 条未来路径。
2. 学生能提出至少 3 个具体信息需求。
3. 学生能写出未来 7 天可执行的一步。
4. 支队能形成至少 3 个交叉验证的问题发现。
5. 至少 3 支其他支队提供横向比较反馈。

### 操作问题：下一步怎么做

1. 出发前完成校方沟通和伦理清单。
2. 到校第一天做基线问卷。
3. 第二到五天穿插课程、夜谈、访谈、路径地图。
4. 第六天做后测和学生反馈。
5. 第七天向学校做匿名化反馈，启动后续机制。

### 表达问题：如何写成优秀实践成果

1. 用一个学生群体的典型困惑开头，但不暴露身份。
2. 用数据说明问题不是单一情绪。
3. 用横向比较说明公共性。
4. 用课程和资源包说明解决机制。
5. 用限制与伦理反思显示克制。

## 四、伦理与安全清单

### 红线

1. 不与未成年人进行封闭一对一深度访谈。
2. 不询问创伤、家庭矛盾、心理疾病、经济隐私等敏感细节，除非学校专业人员在场且有明确必要。
3. 不拍摄学生正脸、宿舍床铺、私人用品、成绩单、聊天记录。
4. 不把学生原话、照片、姓名、班级、家庭情况组合成可识别信息。
5. 不承诺保密到“任何情况都不告诉老师”。若涉及自伤、伤害他人或严重风险，必须启动学校保护机制。

### 夜谈/宿舍观察合规版本

用户提出的“跟高中生一起睡一觉、夜谈”等形式，建议改造为：

1. **宿舍公共空间夜谈**：在学校批准的公共空间或宿舍活动室，6-10 人小组，至少 2 名队员在场，同性别优先，宿管/老师知情且可达。
2. **晚自习后散点圆桌**：不进入寝室内部，在教室、会议室、操场边等开放场所进行。
3. **同宿式生活观察替代方案**：若学校允许队员住校，只观察作息制度、公共空间、晚自习节奏，不记录床位和个人隐私。
4. **睡前心愿瓶**：学生匿名写下“今天最想问但没问出口的问题”，代替深夜高风险访谈。
5. **不做同床、单独陪睡、深夜单独谈话、私人空间拍摄**。

### 学生知情说明简版

你可以自愿参加这个活动。你可以不回答任何不想回答的问题，也可以随时退出。我们只记录匿名内容，不会把你的姓名、班级和个人信息写进报告。这个活动不是考试，不影响成绩，也不会评价你。

## 五、调研工具草案

### A. 8 分钟基线问卷

1. 我能说清自己未来 3 年大概想往哪里走。
2. 我知道实现目标需要哪些具体步骤。
3. 我了解不同大学专业大概学什么、以后能做什么。
4. 我知道遇到升学/专业/学习规划问题可以问谁。
5. 我相信自己可以通过改变方法取得进步。
6. 我现在最大的困惑不是“不想努力”，而是不知道怎么努力。
7. 我的家庭能理解并支持我的升学选择。
8. 我的睡眠、手机或情绪会明显影响学习状态。
9. 我希望大学生哥哥姐姐更多讲真实经历，而不只是鼓励。
10. 如果今天只能问一个关于未来的问题，我想问：____

1. 您觉得学生最大的成长困境是什么？
2. 学生最缺的是学习方法、信息、信心、家庭支持还是时间管理？

## 七、课程设计原则

1. 每节课必须有学生输出物。
2. 每节课最多传达 1 个核心观念。
3. 清华经历只作为案例，不作为标准答案。
4. 避免“你只要努力就能怎样”的简单叙事。
5. 每节课最后 5 分钟收集一个问题或行动承诺。

## 八、后续三个月最小陪伴机制

### 低成本版本

1. 每两周一次线上匿名答疑整理，由学校转发。
2. 建立“问题收集表”，不直接收集学生隐私。
3. 支队每月整理一次《路径问答小报》。
4. 返校后一个月做一次简短追踪问卷。

### 中成本版本

1. 组建 10-20 名自愿学生的“未来路径小组”。
2. 每名队员负责一个主题：专业、学习方法、奖助学金、AI 工具、心理压力资源。
3. 每月一次线上分享或书信回应。

### 高成本版本

1. 对接 SAEPA、研支团、校友、院系科协/实践部。
2. 建立平昌二中长期资源库。
3. 下一年支队接续回访。

## 九、成果形态

### 给学生

1. 《我的三条可达路径》工作纸。
2. 《大学专业与职业入门问答》。
3. 《奖助学金与家庭沟通简明说明》。
4. 《一周行动实验卡》。
5. 后续匿名答疑入口。

### 给学校

1. 匿名化学生需求摘要。
2. 学生最关心的 20 个问题。
3. 支队课程反馈与资源建议。
4. 后续可复用工作纸。

### 给清华实践评优/汇报

1. 故事线：从“支教上课”到“发现未来可达性问题”。
2. 方法线：问卷、夜谈、教师访谈、路径地图、横向比较。
3. 证据线：学生输出、统计结果、主题编码、对照支队反馈。
4. 方案线：资源包、后续机制、学校反馈。
5. 反思线：短期支教边界和未成年人伦理。
