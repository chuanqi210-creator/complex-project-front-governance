current_as_of：2026-06-22

本轮用途：继续把真实项目池扩展到现实时间所有比赛和真实组织项目。本轮选择文化遗产/非遗保护方向，压力测试协议能否区分“宣传展示、体验教育、文创消费、数字传播”和“真正的保护传承、系统性保护、传承人培养、文化生态改善、社区赋能和长期影响”。

## 2. 为什么需要新增 heritage_safeguarding_trace

已有 `aesthetic_cultural_impact_trace` 能处理艺术展演、美育和文化理解，但非遗/文化遗产项目还有更硬的保护边界：

1. 文化遗产不是普通内容素材，涉及项目身份、历史语境、传承谱系、社区认同和保护责任。
2. “看见了、体验了、传播了、买了文创”只能证明触达或消费，不等于传承机制变强。
3. 文旅融合和商业开发有积极空间，也有过度商业化、符号化、误读、挪用和利益分配风险。
4. 数字化和短视频传播可能扩大可见度，但不必然保存技艺、培养传承人或改善文化生态。
5. 真正能说“保护传承”，需要看到保护单位、代表性传承人或社区主体、持续传习、档案记录、教育机制、资金资源、风险控制和后续监测。

## 3. 暴露的失败模式

```yaml
failure_id: heritage_display_gap
symptom: 把文化遗产展演、非遗体验课、研学活动、打卡市集、文创销售、短视频播放、媒体报道、数字化展示或活动照片误当成遗产保护、非遗传承、文化生态改善、社区赋能、代际传承或经济社会发展成效
root_cause: 文化遗产项目的公共可见度很容易被写成保护成效，但展示、传播和消费只是保护传承链条的一部分
trigger_signal:
  - 非遗市集后写“推动非遗保护传承”
  - 短视频播放量高后写“让传统文化活起来”

  - 文创销售后写“实现社区赋能和乡村振兴”

corrected_behavior: 启用 heritage_safeguarding_trace，把结论拆成宣传展示、体验教育、文创消费、数字传播、代表性传承、系统性保护、文化生态和长期影响
detection_check: 搜索最终文本中“保护传承、活化利用、文化认同、社区赋能、代际传承、乡村振兴、经济社会发展”等强结论，逐条要求遗产身份、传承主体、保护责任、文化阐释、授权收益、风险控制和后续监测证据
status: mitigated
```

## 5. 最小结论门

```yaml
minimum_claim_gate:
  publicity_or_display:
    required: 官方通知、活动方案、展示内容、参与主体、受众范围、传播数据或反馈
    downgrade_if_missing: 形成了文化遗产宣传展示输出
  experience_or_education:
    required: 教学/研学目标、传承人或专业人员参与、文化阐释审核、学习反馈、持续课程或复访机制
    downgrade_if_missing: 提供了非遗体验教育机会，学习与传承效果待验证
  cultural_creative_or_consumption:
    required: 授权署名、收益分配、产品来源、文化阐释、质量控制、过度商业化风险评估
    downgrade_if_missing: 形成了文创消费或传播线索，不能称为保护传承成效
  digitization_or_media_spread:
    required: 采集标准、元数据、授权、保存平台、开放边界、误读风险、后续使用计划
    downgrade_if_missing: 形成了数字展示材料或传播素材，不能称为完成保护
  safeguarding_or_transmission:
    required: 遗产项目身份、保护单位或责任主体、传承人/社区参与、持续传习、资源投入、档案记录、评估和后续监测
    downgrade_if_missing: 具备保护传承潜力，仍需持续机制和主体证据
  community_or_development_impact:
    required: 受益群体、收益分配、就业或收入证据、社区反馈、生态和文化风险控制、长期跟踪
    downgrade_if_missing: 可能带来社区或经济效益，实际影响待独立验证
```

## 6. 举例：如何把故事讲实

以下是举例，不是本轮真实发生事实。

一个学生团队拍摄地方传统织造技艺短视频，视频播放量很高，还在市集售卖联名帆布包。最初报告写成：“项目让濒危非遗重回大众视野，实现了活态传承，并带动乡村振兴。”这句话的问题在于，播放量和文创销售只能证明传播与消费，不能证明技艺传承链条被修复、青年学徒稳定成长、传承人收益改善或社区发展。

按 `heritage_safeguarding_trace` 重写后，故事可以变成：

1. 起点：团队原以为让更多人看见非遗，就是保护非遗；实地接触后发现，真正困难是青年学徒少、核心工序难学、传承人收入不稳定、外部文创容易只借用符号。
2. 证据边界：短视频播放量只能证明可见度提升，市集销售只能证明阶段性消费兴趣，不能证明传承问题已经解决。
3. 转化动作：团队与传承人、合作社或保护单位共同确认可公开呈现的工序、名称、署名和收益规则，把视频从“猎奇展示”改成“工序解释 + 学徒招募 + 课程入口 + 授权购买”的组合。
4. 服务闭环：每次体验课记录参与者学习时长、复访意愿和后续报名；文创收益按约定反馈给传承主体；数字素材标注来源和授权，不展示不宜公开或容易误读的核心环节。
5. 可以声明的成果：项目提升了该技艺的公众可见度，形成了经过传承主体确认的传播材料和初步体验课程，并建立了收益回流与后续传习入口。
6. 不能声明的成果：不能写成已经完成非遗保护、已经实现代际传承、已经带动乡村振兴或已经改善文化生态，除非有长期传习、收入、社区反馈和保护计划证据。

这个故事的力量不是拔高，而是承认“看见”只是第一步：真正的公共价值在于从曝光走向尊重主体、准确阐释、收益回流和持续传习。
