## 二、为什么这一类项目会暴露新缺口

体育项目的成果非常容易被竞技指标替代：参赛了、赢了球、进入八强、拿了奖牌、刷新纪录、场馆上座率高、赛事报道热烈。它们可以证明竞赛 output 或高水平竞技表现，却不能自动证明普通学生体质健康提升、运动技能普及、终身锻炼习惯养成、体育精神或团队协作改善。

学校体育的公共目标比竞赛结果更宽。中办、国办学校体育意见强调“健康第一”、日常参与、体质监测、专项运动技能测试和“教会、勤练、常赛”。因此，体育类项目需要一条专门的证据链：赛事输出只是起点，体育育人结论必须回到参与覆盖、基线、训练负荷、体质/技能测评、日常运动、伤病与恢复、安全管理和持续机制。

## 三、旧流程可能误判

如果旧流程只看到“全国大学生排球联赛”“学体联”“高水平赛事”“主客场”“上座率”“八强”“冠军”等关键词，容易产生三类误判：

1. 把校队或少数运动员成绩写成“全校学生体质提升”。
2. 把赛事热度和上座率写成“大学生体育氛围和运动习惯显著改善”。
3. 把高强度训练和竞赛经历写成“锤炼意志、健全人格”，但没有安全、伤病、恢复和学业负荷边界。

这些说法不是一定不能写，而是必须有更高层级证据：目标人群是谁，普通学生还是校队；参与覆盖多少；是否有基线和后测；是否有日常锻炼数据；是否记录运动技能变化、体测指标、伤病和恢复；是否形成持续课程、社团、联赛或训练机制。

## 五、最小结论门

| 想写的结论 | 最低证据要求 | 缺证据时的降级表述 |
| --- | --- | --- |
| 完成了体育赛事或训练活动 | 官方通知、赛程、参赛名单、比赛记录、训练记录或活动照片 | “形成了体育赛事/训练活动输出” |
| 提升了参与覆盖 | 目标总体、参与人数、参与比例、参与频次、性别/年级/资源差异 | “扩大了体育活动触达范围，参与质量待评估” |
| 提升了运动技能 | 技能目标、训练设计、教练指导、前后测或比赛技术统计 | “提供了专项技能训练机会，技能提升待测评验证” |
| 提升了体质健康 | 体测基线、体测后测、持续周期、对照或合理比较、伤病记录 | “具备体质改善潜力，实际变化待持续监测” |
| 养成运动习惯 | 日常运动频次、持续时长、课内外参与、社团/联赛持续机制 | “激发了阶段性运动参与，习惯养成仍需长期证据” |
| 形成体育精神或团队协作 | 反思材料、同伴/教练评价、行为证据、冲突处理、逆境经验和安全边界 | “提供了团队竞赛体验，育人成效需进一步材料支撑” |

## 六、可用于后续项目报告的高质量举例

以下是举例，不是本轮已经发生的事实：

假设一个学院排球队在大学生联赛中进入全国八强，最初准备在汇报中写“显著提升了学院学生体质健康，营造了全民体育氛围”。按新 trace，这句话不能直接成立。八强成绩能证明校队竞技表现，不能证明全院学生体质提升；现场观众多能证明赛事关注度，不能证明观众形成运动习惯。

如果学院进一步把联赛经验转化为排球公共课、低门槛社团训练、女生友好训练时段、零基础技能营和院内联赛，并记录参与人数、每周运动频次、技能前后测、体测变化、伤病处理和学生反思，才可以谨慎写“以高水平赛事带动了更广泛的排球参与和阶段性体育学习”。如果要写“体质健康提升”或“终身锻炼习惯养成”，还需要更长周期的体测和日常参与数据。

这个例子的价值在于：故事不是“我们赢了比赛，所以体育育人成功”，而是“我们先以为成绩就是影响，后来发现成绩只是入口；真正有公共价值的是如何把少数人的竞技经验转化为多数人的日常参与、技能学习和安全持续机制”。

## 八、对低摩擦入口的影响

用户不需要说“请启用 physical_education_impact_trace”。只要出现“体育赛事、学校体育、运动会、联赛、锦标赛、训练营、体育课程、体育社团、课后训练、体测、体质健康、运动技能、运动习惯、体育精神、团队协作、健康生活方式”等自然语言，AI 就应自动触发。

这个模块与 `learning_outcome_trace` 的边界是：如果项目主要证明具体课程学习目标，优先学习成效；如果项目要证明体质健康、运动技能、运动习惯、体育精神或学校体育机制，启用本模块。与 `biomedical_health_claim_trace` 的边界是：如涉及疾病治疗、康复、医疗安全或临床健康主张，应另启生命健康主张追踪。
