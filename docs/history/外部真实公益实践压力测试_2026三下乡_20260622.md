## 一、为什么选择这个项目

本轮从公益志愿、社会实践或创新创业渠道选择真实项目，避免上一轮外部项目仍停在 AI/数据比赛。

选择 2026 年全国大学生文化科技卫生“三下乡”社会实践活动通知的原因：

1. 它是 2026 年现实时间的全国性社会实践项目通知。
2. 它由中央宣传部、教育部、共青团中央、全国学联组织开展，来源层级高。
3. 它覆盖乡村振兴、科技创新、社会服务、绿色发展、文化传承、调研报告、揭榜攻关等多类实践形态。
4. 它直接暴露“社会实践活动完成”与“真实社会影响”之间的断层。
5. 它与用户最初的支教/县中实践项目同属社会实践语境，但范围更大，可以反过来校正平昌支教方案。

## 三、该项目暴露的协议缺口

### 缺口 1：活动完成不等于社会影响

社会实践最容易把“去了、讲了、拍了、写了”当成成果。

但通知中的“实效长效”和“五个一”要求提示，实践项目至少应区分：

1. 活动输出：宣讲次数、课程场次、调研报告、视频照片。
2. 服务对象反馈：学生、村民、基层干部、教师、合作单位的实际评价。
3. 需求匹配：是否来自基层真实需求，而不是支队单方面想象。
4. 结果证据：惠民实事是否完成，建议是否被反馈/采纳，结对是否延续。
5. 长效机制：是否有后续联系人、材料交接、复访时间和责任人。

### 缺口 2：调研报告不等于资政价值

通知强调“小切口、深挖掘、重转化”，这意味着调研报告不能只写宏大意义。

至少要追踪：

1. 问题是否来自现场反复出现的真实困境。
2. 结论是否有访谈、问卷、观察、材料证据。
3. 建议是否能被地方或学校执行。
4. 是否反馈实践地供其参考。
5. 哪些结论只能作为案例，不足以推广为公共问题。

### 缺口 3：公益实践必须把安全、伦理和报备当成硬边界

社会实践不是纯文本项目。它涉及学生、乡村、未成年人、基层组织、宣传和舆情。

必须记录：

1. 学校/组织报备和审核状态。
2. 服务对象同意、未成年人保护、隐私和影像授权。
3. 出行、住宿、医疗、天气、交通、突发事件预案。
4. 宣传材料是否夸大成效或暴露个人隐私。
5. 实践地是否认可项目表述。

## 五、最小结论门

社会实践、支教、公益志愿或乡村项目，在最终报告中写“产生影响”“解决问题”“助力发展”“形成机制”之前，必须至少满足：

1. 有来自服务对象或地方伙伴的需求来源。
2. 有活动输出以外的反馈或结果证据。
3. 有最低影响标准，而不是只有照片、推文、课程表或心得。
4. 有后续承接责任人、材料交接或复访机制。
5. 有安全、伦理、隐私、报备和宣传边界记录。

## 六、协议修正目标

本轮应写回协议的新约束：

1. 社会实践、支教、公益志愿、乡村振兴和公共服务项目，必须维护 `service_impact_trace`。
2. 新失败模式命名为 `activity_impact_gap`：把活动输出、宣传材料、调研报告或团队感受误当成真实社会影响。
3. 执行前闸门新增：是否区分活动输出、服务对象反馈、地方需求、结果证据、长效机制和安全伦理边界。
