## 二、为什么选择这个样本

前几轮已覆盖 AI 数据赛、公益实践、创业、硬件、开源、数值计算和数学建模，但还缺少一类真实项目：传播、设计、广告、内容创作、公众说服和品牌/公益倡导。

这类项目在学生实践中很常见，也与支教、社会实践、公共议题汇报高度相关。它的风险不是“没有做出东西”，而是：

1. 作品完整，不等于目标受众真的理解、相信或行动。
2. 播放量、点赞和平台综合数据，不等于品牌认知、购买意向、公共意识或社会态度改变。
3. 专家评审认可，不等于真实传播场景有效。
4. AIGC 作品完成，不等于版权、原创性、事实准确性和价值导向风险已处理。
5. 公益传播或公共议题表达动人，不等于被表达对象真正受益，也不等于社会问题被解决。

## 三、本轮暴露的新协议缺口

现有 `service_impact_trace` 能处理公益服务影响，`business_model_validation_trace` 能处理商业验证，`competition_to_deployment_trace` 能处理榜单到部署，但还缺少一个直接处理“传播作品是否有效”的追踪。

例如一支社会实践队可能做出一条“县中学生如何面对 AI 时代”的短视频：

1. 采访素材真实，剪辑流畅，故事很感人。
2. 发布后获得较多播放量和点赞。
3. 汇报时被认为“有传播力、有社会影响”。

但如果没有说明目标受众是谁、视频想改变什么认知、观看者是否真的理解、是否有误读、是否对学生造成标签化、是否有后续行动入口，就不能声称“该传播改善了县中学生对 AI 的认知”。更稳妥的结论只能是“该作品在某平台上获得了一定可见度，并提出了一个值得继续验证的公共议题表达”。

## 五、最小结论门

传播设计类项目只有同时满足以下条件，才能把作品上升为较强传播效果结论：

1. 传播目标清楚：是知晓、理解、态度、信任、转化、参与，还是公共议题讨论。
2. 目标受众清楚：不是笼统“大众”，而是可描述的人群和场景。
3. 作品与目标、受众和渠道匹配，而不是只凭美感或故事性。
4. 至少区分 output、outcome 和 impact：发布数量/播放量是产出，不自动等于认知或行为改变。
5. 至少有一种受众反馈或测试方法：小样本访谈、问卷、评论编码、A/B 测试、品牌提升、转化数据或行动反馈。
6. AIGC、素材来源、肖像、字体、音乐、商标、平台规则和广告法/公益伦理风险有处理记录。
7. 明确写出不能外推的场景。

如果上述条件不足，最终报告必须降级：

| 不能写 | 应降级为 |
| --- | --- |
| 作品产生了广泛社会影响 | 作品获得了一定传播可见度 |
| 播放量证明受众被说服 | 播放量只能证明触达或平台分发，不能直接证明态度改变 |
| 该公益传播解决了认知问题 | 该公益传播提出了议题表达，需要受众反馈继续验证 |
| AIGC 作品可直接商用 | AIGC 作品需完成版权、事实、平台和品牌安全核验 |

## 六、对前置协议的修正

新增失败模式：

```yaml
agent_failure_modes:
  - failure_id: F44-01
    name: creative_effect_gap
    symptom: 把作品完成、视觉质量、播放量、点赞、平台综合数据、专家评分或获奖误当成真实传播效果、品牌转化或公共认知改变
    root_cause: 只追踪作品和曝光，没有追踪传播目标、目标受众、outcome、impact、归因边界、合规伦理和受众反馈
    trigger_signal:
      - 广告、视觉设计、短视频、直播、AIGC 作品、公益传播、品牌传播或公共议题传播项目
      - 结论中出现“传播力强、影响广泛、改变认知、带来转化、形成社会影响”等强表述
      - 用播放量、点赞、专家评审或获奖支撑传播效果
    corrected_behavior: 启用 communication_effect_trace，并把传播结论降级到证据能够支持的层级
    detection_check: 桌面协议应能检索到总规则 41、执行闸门 100、communication_effect_trace 和 creative_effect_gap
    regression_case: R44-communication-effect-trace
    status: mitigated_with_trace_module
```

新增回归样例：

```yaml
agent_regression_cases:
  - case_id: R44-communication-effect-trace
    user_prompt_pattern: 用户要求做广告、传播、短视频、直播、AIGC作品、公益倡导、品牌策划或公共议题传播，并希望证明有影响力
    expected_behavior: 先核验真实命题或项目来源，再启用 communication_effect_trace，区分作品产出、触达数据、受众结果、潜在影响和归因边界
    forbidden_behavior:
      - 只给创意方向和作品清单
      - 把播放量、点赞、专家评审或获奖当成真实传播效果
      - 不说明目标受众、合规伦理、AIGC 来源和不能外推范围
    required_tools_or_checks:
      - live_project_pool_scan
      - source_link_recency_audit
      - trace_module_router
      - communication_effect_trace
    pass_condition: 最终表达中能清楚看到作品产出、受众反应、可验证结果、不能声称的影响和下一步验证方法
    last_tested: 2026-06-22
```

## 七、对用户低摩擦入口的影响

用户不需要说“communication_effect_trace”。只要出现以下自然语言，AI 应自动触发：

1. “广告、传播、短视频、直播、AIGC作品、公益宣传、品牌策划、公共议题传播、让更多人看到/相信/行动”。
2. “证明这个作品有影响力/有传播力/能改变认知/能带来转化”。
3. “做一套对外传播材料、海报、推文、视频、路演传播方案”。

用户可见回复只需要说明：

“这是传播/设计/公众说服类项目，我会额外检查目标受众、传播目标、受众反馈、合规伦理和效果归因，避免把作品完成或播放量外推成真实影响。”
