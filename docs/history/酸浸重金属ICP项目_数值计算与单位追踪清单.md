# 酸浸重金属 ICP 项目：数值计算与单位追踪清单

用途：为酸浸重金属 ICP 稀释方法比较、除 As 与 FePO4·2H2O 制备方案建立数值追踪层，防止论文或实验方案中的精确数字失去来源、单位、公式和适用边界。

使用时间：写论文前、更新示范参数后、替换真实 ICP/密度/器具证书数据前、生成 Word/PDF 前。

## 一、数值身份规则

所有数字先分成四类，不能混写：

| 类型 | 定义 | 写法要求 |
| --- | --- | --- |
| 实测值 | 来自 ICP、密度、天平、移液器校准、重复实验 | 必须记录原始文件、单位、日期、仪器/器具、样品编号 |
| 证书/规格值 | 来自容量瓶、移液器、天平、标准液证书或方法文件 | 必须记录证书编号、允差分布假设、是否仍在有效期 |
| 示范参数 | 为论文模型或预算展示假设的数值 | 必须标注“示范”，不能写成真实实验结果 |
| 派生计算值 | 由公式从输入量计算得到 | 必须记录公式、输入量、单位换算和复算状态 |

| 计算主张 | 输入量 | 公式/换算 | 复算结果 | 状态 | 不能说明什么 |

## 三、ICP 稀释论文必须追踪的数值链

| 链条位置 | 必填字段 | 高风险错误 |
| --- | --- | --- |
| 原液浓度 | 样品编号、元素、单位、稀释前/后、空白扣除方式 | 把最终上机液浓度当成原液浓度 |
| 稀释操作 | 每步转移量、终体积/总质量、步数、目标稀释倍数 | 把 `1:10000` 写成单步但实际按连续稀释计算 |
| 质量稀释密度修正 | `rho_0`、`rho_n`、温度、密度测量方法 | 只用质量比，却报告体积浓度 |
| 不确定度输入 | 允差、分布、重复性、证书来源、ICP 项 | 把示范预算写成真实不确定度 |
| 绿色指标 | 新增稀释液体积、酸性废液体积、容器数、操作步数 | 只写“减少 99%”，不说明基准链条 |
| 结论边界 | Boundary 1/2/3/4 | 用稀释项结论替代完整 ICP 结果结论 |

## 四、替换真实数据前的过门清单

1. 所有示范参数是否仍标注为“示范”？
2. 每个真实 ICP 数值是否能回到原始记录或仪器导出？
3. `mg/L`、`ug/L`、`mg/kg`、`g/L` 是否逐项标注？
4. 稀释倍数是 `D = V_transfer / V_final` 还是放大因子 `F = V_final / V_transfer`？全文是否统一？
5. 质量稀释是否说明 ICP 体积浓度反算中的密度项？
6. 每个结论使用的是 Boundary 1、Boundary 2、Boundary 3 还是 Boundary 4？
7. 绿色指标是否和同一目标稀释倍数、同一步数的基准比较？
8. Word/PDF 图表是否由最新 Markdown 或计算表生成？
