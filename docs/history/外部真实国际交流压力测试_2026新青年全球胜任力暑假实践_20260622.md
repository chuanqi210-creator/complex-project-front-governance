## 二、项目事实

中国教育国际交流研修学院官网通知显示，2026 年度暑假实践活动意在推进“新青年全球胜任力人才培养计划”的课程与实践结合，活动类别包括海外实习实践、文化交流、外交访问、名校访学等。通知列出 16 类活动，例如国际志愿者实习项目、国际组织实习生项目、国际企业实习项目、青年使者赴美交流项目、联合国纽约总部与哥伦比亚大学实地交流项目、联合国日内瓦/维也纳/亚太总部实地交流项目、香港学术探索项目、伦敦大学学院神经科学践习项目、昆士兰大学国际医护体系与临床实践项目等。

这说明项目真实存在、项目形态足够复杂，也说明它会同时触发语言能力、跨文化理解、国际组织认知、全球议题学习、海外实践安全和合作持续性等主张边界。

## 四、本轮故事线

我们一开始容易被这类项目的“国际资源密度”吸引：联合国总部、海外名校、国际组织、国际企业、青年使者、志愿服务，这些词都非常适合写成漂亮汇报。

真正的问题是：这些词本身只说明学生获得了跨境接触机会，不说明学生已经形成可迁移的全球胜任力。全球胜任力不是“去过哪里”的履历标签，而是学生能不能理解复杂议题、看见他者视角、在差异中协作，并把这种理解转化成负责任行动。

所以，协议必须把国际交流类项目从“活动完成叙事”拉回“能力与合作证据”：出访和交流是输入，证书和合影是过程材料，真正要证明的是学生在语言、视角、协作、反思、伦理意识和持续合作上的变化。

## 五、新失败模式

```yaml
failure_mode:
  id: F57-01
  name: exchange_visit_outcome_gap
  symptom: 项目报告把出访、接待、合影、证书、课程出勤、交流场次、MOU、社交媒体传播或满意度写成全球胜任力提升、跨文化理解、语言能力提升、实质国际合作、公共外交成效或长期伙伴关系形成。
  root_cause: 把国际交流的机会、资源和符号当作能力结果，没有区分 exposure、output、outcome、impact。
  trigger_signal:
    - 用户说全球胜任力、国际视野、跨文化能力、国际理解、公共外交、国际合作、海外实践、外事交流、国际组织实习、联合国项目。
    - 材料里主要证据是项目通知、照片、证书、行程、课程表、满意度和新闻稿。
  corrected_behavior: 启用 international_exchange_impact_trace，先记录项目事实，再要求基线、能力目标、互惠设计、共同任务、联合产出、评价证据、伙伴反馈、安全伦理和后续合作；证据不足时降级为交流机会或待验证假设。
  detection_check: 报告中每一个“能力提升/合作成效/公共影响”主张都必须能落到至少一个非符号性证据；若只能落到出访或活动材料，则必须降级。
```

## 六、international_exchange_impact_trace

```yaml
international_exchange_impact_trace:
  project_name: 2026年度暑假新青年全球胜任力人才培养计划实践活动
  source_url: https://www.ciei.org.cn/sys-nd/708.html
  current_as_of: 2026-06-22
  exchange_or_global_competence_claim: 不能默认成立；需按具体院校、具体学生和具体活动证据判断
  project_type: 海外实习实践 / 文化交流 / 外交访问 / 名校访学 / 国际组织实地交流
  participating_countries_or_regions: 美国、瑞士、奥地利、亚太地区、法国、比利时、荷兰、香港、英国、澳大利亚等通知列示方向
  partner_institutions_and_roles: 待具体活动方案和合作方确认；官网通知只能证明活动方向和组织安排
  participant_selection_and_baseline: 需记录学生年级、专业、英语水平、既有国际经历、全球议题认知和参与动机
  language_and_intercultural_baseline: 需有前测、面试、作品、反思日志或任务表现，不能只写“具备英语能力”
  exchange_design_duration_and_intensity: 需记录时长、课程/参访/实习任务比例、真实互动时长、语言环境和任务难度
  reciprocity_and_mutual_benefit_design: 需说明当地伙伴、受访机构或共同参与者是否不仅是被参访对象，而是共同任务参与方
  collaboration_tasks_and_joint_outputs: 需有联合报告、案例分析、访谈纪要、政策简报、双语展示、项目提案或共同完成的作品
  learning_or_competence_assessment: 需有前后测、rubric、导师评价、伙伴评价、同伴互评或学生作品对比
  language_growth_evidence: 需有口头展示、写作任务、访谈表现、专业词汇使用或真实沟通反馈
  intercultural_reflection_evidence: 需有结构化反思，说明学生如何修正刻板印象、理解制度差异、处理价值冲突和沟通误解
  partner_feedback_and_external_validation: 需有海外机构、导师、伙伴学生、服务对象或合作方反馈
  cultural_sensitivity_ethics_and_consent: 需说明拍摄、访谈、调研、志愿服务、公众传播是否获得授权并避免符号化他者
  accessibility_equity_safety_and_wellbeing: 需记录费用、选拔公平、保险、健康安全、突发事件、心理支持和退出机制
  visa_travel_data_privacy_and_risk_management: 需记录签证、行程、数据、照片、访谈材料和个人信息处理边界
  follow_up_collaboration_or_network: 需有返校分享、课程转化、后续共同项目、持续联络或二次合作证据
  public_communication_or_diplomacy_boundary: 对外传播只能证明表达和触达，不能自动证明公共外交效果
  outcome_attribution_boundary: 能力变化受学生基础、课程训练、海外环境和后续实践共同影响，不能把变化全部归因于一次出访
  evidence_to_exchange_claim_mapping:
    project_notice: 可证明项目真实存在和活动形态
    itinerary_or_photo: 可证明参与和接触，不证明能力变化
    certificate: 可证明完成，不证明掌握
    reflection_and_assessment: 可部分证明理解变化
    partner_feedback_and_joint_output: 可支持跨文化协作主张
    follow_up_project: 可支持持续合作主张
  cannot_claim:
    - 不能凭出访经历声称全球胜任力显著提升
    - 不能凭联合国或名校参访声称具备国际组织职业能力
    - 不能凭交流场次声称形成长期伙伴关系
    - 不能凭社交媒体传播声称公共外交成效
  replacement_condition: 缺少能力证据时，写成“提供了国际交流机会、跨文化接触和全球议题学习场景；能力变化与合作影响需通过前后测、共同产出、伙伴反馈和后续合作继续验证”
```

## 七、最小结论门

```yaml
minimum_claim_gate:
  international_exposure:
    required: 项目通知、名单、行程、参与记录、安全与合规材料
    downgrade_if_missing: 国际交流线索或报名机会，参与事实待确认
  intercultural_learning:
    required: 学习目标、前测、课程/互动任务、结构化反思、后测或作品变化
    downgrade_if_missing: 跨文化接触体验，不能称为跨文化能力提升
  language_improvement:
    required: 语言基线、真实沟通任务、口头或书面作品、外部反馈、前后变化
    downgrade_if_missing: 英语使用场景，不能称为语言能力提升
  collaboration_output:
    required: 共同任务、双方角色、联合产出、伙伴反馈、修订记录
    downgrade_if_missing: 交流活动或合作意向，不能称为实质协作成果
  public_diplomacy_or_cultural_communication:
    required: 目标受众、传播内容、反馈证据、误读风险控制、伦理授权
    downgrade_if_missing: 对外表达或文化展示，不能称为公共外交成效
  long_term_partnership:
    required: 后续项目、持续联络、制度化合作、资源投入、阶段性结果
    downgrade_if_missing: 合作意向或一次性接触，不能称为长期伙伴关系
```
